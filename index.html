DOM 조작

DOM (Document Object Model)은 HTML 문서의 구조를 표현하는 객체 모델입니다.




DOM 선택자 종류

document.getElementById -> id로 요소를 선택

document.getElementsByClassName -> class로 요소를 선택

document.getElementsByTagName -> 태그명으로 요소를 선택

document.querySelector -> CSS 선택자로 요소를 선택

document.querySelectorAll -> CSS 선택자로 여러 요소를 선택



DOM 수정

textContent -> 요소의 텍스트 내용을 변경

innerHTML -> 요소의 HTML 내용을 변경

setAttribute -> 요소의 속성 값을 변경

style -> 요소의 스타일을 변경



DOM 추가/삭제

createElement -> 새로운 요소를 생성

appendChild -> 요소를 부모 요소에 추가

removeChild -> 부모 요소에서 자식 요소를 삭제

replaceChild -> 부모 요소에서 자식 요소를 교체
